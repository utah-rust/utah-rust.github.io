Editing the Website

Install Rust:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Install Cobalt:

cargo install cobalt-bin

Clone the repo and enter website:

git clone git@github.com:utah-rust/utah-rust.github.io.git
cd ./utah-rust.github.io/

Build the website:

cobalt build

Preview:

cobalt serve

The website is built using cobalt and liquid and is hosted using Github Pages.

  1. Get started with cobalt
  2. Edit and create a PR for the source branch (master is for the built site).
